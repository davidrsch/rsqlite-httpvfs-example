mtcars.sqlite

This GitHub Pages site hosts a small SQLite database file for HTTP read-only demos (RSQLite HTTP VFS).

  • mtcars.sqlite
  • mtcars.sqlite.sha256 (optional integrity check)

Use from R

Example (requires an RSQLite build with HTTP VFS support):

library(DBI)
library(RSQLite)

url <- "https://<user>.github.io/<repo>/mtcars.sqlite"

stopifnot(sqliteHasHttpVFS())
con <- sqlite_remote(url)
dbGetQuery(con, "SELECT COUNT(*) AS n FROM mtcars")
dbDisconnect(con)
